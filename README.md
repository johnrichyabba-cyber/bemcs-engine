# BE-MCs Engine
Bandari Electronic Monitoring and Control System

## Description
This project is a shipment tracking, registry, accounting, and control dashboard built with Node.js and Express.

## Features
- Login system
- Dashboard with alerts and notifications
- Tracking module
- Shipment registry
- Registry detail tabs
- Accounting linkage
- Document attachments
- Audit log
- Global search
- Role-based widgets

## Default Login Accounts
- admin / 1234
- operations / 1234
- accounts / 1234
- customs / 1234
